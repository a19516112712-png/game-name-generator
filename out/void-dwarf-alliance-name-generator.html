Rule 1: The name must reflect the Void theme. Every dwarf alliance name incorporates void elements in its prefix or root word, establishing the alliance's magical or elemental alignment.

Rule 2: Honor Dwarf naming traditions. Dwarf names follow specific phonetic patterns — certain consonant clusters, vowel preferences, and syllable structures that distinguish dwarf names from those of other races.

Rule 3: Match the Alliance scale. Empire names should sound grand and sweeping; guild names should feel tight-knit and purposeful; clan names should evoke kinship and shared heritage.
