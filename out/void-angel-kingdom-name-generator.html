Naming Rules for Void Angel Kingdom

Rule 1: The name must reflect the Void theme. Every angel kingdom name incorporates void elements in its prefix or root word, establishing the kingdom's magical or elemental alignment.

Rule 2: Honor Angel naming traditions. Angel names follow specific phonetic patterns — certain consonant clusters, vowel preferences, and syllable structures that distinguish angel names from those of other races.

Rule 3: Match the Kingdom scale. Empire names should sound grand and sweeping; guild names should feel tight-knit and purposeful; clan names should evoke kinship and shared heritage.

Rule 4: Maintain internal consistency. If you generate multiple angel kingdoms for the same world, ensure their names share linguistic DNA while remaining distinct enough to avoid confusion.
