Naming Rules for Shadow Angel Order

Rule 1: The name must reflect the Shadow theme. Every angel order name incorporates shadow elements in its prefix or root word, establishing the order's magical or elemental alignment.

Rule 2: Honor Angel naming traditions. Angel names follow specific phonetic patterns — certain consonant clusters, vowel preferences, and syllable structures that distinguish angel names from those of other races.

Rule 3: Match the Order scale. Empire names should sound grand and sweeping; guild names should feel tight-knit and purposeful; clan names should evoke kinship and shared heritage.

Rule 4: Maintain internal consistency. If you generate multiple angel orders for the same world, ensure their names share linguistic DNA while remaining distinct enough to avoid confusion.
