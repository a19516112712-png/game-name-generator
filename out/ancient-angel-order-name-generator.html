Naming Rules for Ancient Angel Order

Rule 1: The name must reflect the Ancient theme. Every angel order name incorporates ancient elements in its prefix or root word, establishing the order's magical or elemental alignment.

Rule 2: Honor Angel naming traditions. Angel names follow specific phonetic patterns — certain consonant clusters, vowel preferences, and syllable structures that distinguish angel names from those of other races.

Rule 3: Match the Order scale. Empire names should sound grand and sweeping; guild names should feel tight-knit and purposeful; clan names should evoke kinship and shared heritage.
